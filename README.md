**53% of Indian women avoid routes due to safety fears. 2.7 crore disabled citizens have no accessible navigation. 4.5 lakh road accidents happen every year.** Google Maps knows none of this. It just tells you to turn left.

StreetSmart is the navigation layer India has always needed — one that scores every route on safety, accessibility, and real-time ground conditions, not just time and distance.

---

## How it works

![StreetSmart Intelligent Route Generation Flowchart](Smart_route_generation_flowchart_diagram.png)

User inputs destination and preference (fastest / safest / accessible) → AI pulls live traffic, crime, crowd density, and infrastructure data simultaneously → TensorFlow scores each candidate route 0–100 for safety → best route is served with live hazard alerts and dynamic re-routing. In parallel, a civic feedback loop lets citizens report issues in under 10 seconds, which are AI-validated and pushed to authorities — and fed back into future routing.

## What we built

| Capability                   | What it does                                                                                                 |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------ |
| 🛡️ **Safety routing**        | Scores routes using crime data, accident history, and lighting levels. Flags danger zones in real time.      |
| ♿ **Accessibility-first**   | Wheelchair paths, ramp metadata, elevator availability, audio guidance for visually impaired users.          |
| 🚦 **Predictive traffic**    | TensorFlow model forecasts congestion **30 minutes ahead** — reroutes before you're stuck.                   |
| 👥 **Civic feedback loop**   | Citizens report potholes, broken lights, unsafe zones → AI validates → authorities alerted → routes updated. |
| 🚨 **Emergency integration** | One-tap routing to nearest hospital or police station. Live location sharing with emergency contacts.        |
